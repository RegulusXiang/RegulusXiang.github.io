海子诗抄

2016-05-09

四个月，终究静下心读完了《海子诗全编》的短诗部分，摘录了一些我喜欢的句子。

短诗：1983-1986

 

【亚洲铜】

爱怀疑和飞翔的是鸟，淹没一切的是海水

你的主人却是青草，住在自己细小的腰上，

守住 野花的手掌和秘密

 

【阿尔的太阳——给我的瘦哥哥】

把星空烧成粗糙的河流

把土地烧得旋转

 

【海上】

成群游来鱼的脊背

无始无终

只有关于青春的说法

一触即断

 

【我，以及其他的证人】

那些寂寞的花朵

是春天遗失的嘴唇

 

【单翅鸟】

穿过自己的手比穿过别人的墙壁还难

 

【跳跃者】

我走过许多条路

我的袜子里装满了错误

 

【印度之夜】

城市，最近才出现的小东西

跟沙漠一样爱吃植物和小鱼

 

【不要问我那绿色是什么】

我避开所有的道路

最后长成

站在风熏寓言的石墓上 长成

不要问我那绿色是什么

 

【黑风】

人们啊，所有交给你的

都异常沉重

你要把泥沙握得紧紧

在收获时应该微笑

没必要痛苦地提起他们

没必要忧伤地记住他们

 

【龙】

远方就是你一无所有的地方

 

【女孩子】

春天是风

秋天是月亮

在我感觉到时

她已去了另一个地方

 

【海上婚礼】

风吹起你的

头发

一张棕色的小网

撒满我的面颊

我一生也不想挣脱

 

【坛子】

额头披满钟声的

土地

一只坛子

 

【活在珍贵的人间】

活在这珍贵的人间

太阳强烈

水波温柔

…

 

活在这珍贵的人间

人类和植物一样幸福

爱情和雨水一样幸福

 

【夏天的太阳】

你来人间一趟

你要看看太阳

 

和你的心上人

一起走在街上

 

【北方门前】

我愿意

愿意像一座宝塔

在夜里悄悄建成

 

晨光中她突然发现我

她眺起眼睛

她看得我浑身美丽

 

【写在脖子上的菩萨】

两片抖动的小红帆

含在我的唇间

…

 

就让我出生

让我长成的身体上

挂着潮湿的你

 

【哑脊背】

他水国的口音很重

 

【我请求：雨】

雨是一生过错

雨是悲欢离合

 

【早祷与枭（组诗）】

太阳太远了

否则我要埋在那里

 

【明天醒来我会在哪一只鞋子里】

我不声不响地

带来自己这个包袱

尽管我不喜爱自己

但我还是悄悄打开

 

【夜月】

太阳把血

放入灯盏

 

【给母亲（组诗）】

风 吹遍草原

马的骨头 绿了

…

 

妈妈，妈妈

你面朝谷仓

脚踩黄昏

我知道你日渐苍老

 

【九盏灯（组诗）】

我轻轻走过去关上窗户

我的手扶着自己 像清风扶着空空的杯子

我摸黑坐下 询问自己

杯中幸福的阳光如今何在？

 

我脱下破旧的袜子

想一想明天的天气

 

我的名字躺在我身边

像我重逢的朋友

我从没像今夜这样珍惜自己

 

【村庄】

珍惜黄昏的村庄，珍惜雨水的村庄

万里无云如同我永恒的悲伤

 

【坐在纸箱上想起疯了的朋友】

地震时天空很安全

伴侣很安全

喝醉酒时酒杯很安全

心很安全

 

【在昌平的孤独】

孤独是一只鱼筐

是鱼筐中的泉水

放在泉水中

…

 

孤独不可言说

 

【马（断片）】

整座城市被我的创伤照亮

 

【爱情诗集】

不知道何时献上

不知道怎样安放

 

【歌或哭】

你说你孤独

就像很久以前

长星照耀十三个州府

的那种孤独

 

【幸福（或我的女儿叫波兰）】

在有太阳的弯曲的木头上

晒干你美如黑夜的长发

 

【葡萄园之西的话语】

你这女子中极美丽的，你是我的棺材

我是你的棺材

 

【感动】

早晨是一只花鹿

踩到我额上

世界多么好

…

 

野花烧到你脸上

把你烧伤

世界多么好

 

【黎明】

那是在五月，黎明之前的深水杀死了我

 

【给萨福】

用一只嘴唇摘取另一只嘴唇

…

 

你装饰额角的诗歌何其甘美

你凋零的棺木像一盘美丽的

棋局

 

【给卡夫卡】

在冬天放火的囚徒

无疑非常需要温暖

 

【莫扎特在《安魂曲》中说】

请整理好我那零乱的骨头

放入那暗红色的小木柜，带回它

像带回你们富裕的嫁妆

 

【泪水】

群山似穷孩子的灰马和白马

在十月的最后一夜

倒在血泊中

…

 

在十月的最后一夜

我从此不再写你

 

【七月的大海】

我们会把幸福当成祖传的职业

放下手中痛苦的诗篇

…

 

在七月我总能突然回归到荒凉

 

【给你（组诗）】

我相信有人正慢慢地艰难地爱上我

别的人不会，除非是你

 

【哭泣】

哭泣——一朵乌黑的新娘

我要把你放在我的床上

我的泪水中有对自己的哀伤

 

【怅望祁连（之二）】

“亡我祁连山，使我牛羊不蕃息

失我胭脂山，使我妇女无颜色”

 

【九月】

目击众神死亡的草原上野花一片

远在远方的风比远方更远

…

 

远方只有在死亡中凝聚野花一片

明月如镜高悬草原映照千年岁月

 

【喜马拉雅】

高原悬在天空

天空向我滚来

我失去了一切

面前只有大海

…

 

饥饿用粮食喂养

更加饥饿，奄奄一息

 

短诗：1987-1989

 

【雨】

阳光和雨水只能给你尘土和泥泞

你在伞中，躲开一切

拒绝泪水和回忆

 

【雨鞋】

我则把这些温暖的灰烬

把这些信塞进一双小雨鞋

让她们沉睡千年

梦见洪水和大雨

 

【吊半坡并给擅入都市的农民】

自由的血也有死亡的血

智慧的血也有罪恶的血

 

【九首诗的村庄】

大地在耕种

一语不发，住在家乡

像水滴、丰收或失败

住在我心上

 

【两座村庄】

风吹在村庄的风上

有一阵新鲜有一阵久远

 

【日出——见于一个无比幸福的早晨的日出】

我是一个完全幸福的人

我再也不会否认

我是一个完全的人 我是一个无比幸福的人

 

【不幸（组诗）——给荷尔德林】

血以后是黑暗——比血更红的是黑暗

 

【美丽白杨树】

在黄金和允诺的地上

陪伴花朵和诗歌 静静地开放 安详地死亡

 

【北方的森林】

大地孑然一身 至今仍孑然一身

…

 

麦田吹来微风 顷刻沉入黑暗

 

【月光】

不要说死亡的烛光何须倾倒

生命依然生长在忧愁的河水上

月光照着月光 月光普照

今夜美丽的月光合在一起流淌

 

【灯】

灯光，我明丽又温暖

的橘黄的雪

披上新娘的微黄的发辫

 

【夜晚 亲爱的朋友】

大地茫茫，河水流淌

是什么人掌灯，把你照亮

 

【晨雨时光】

当众人齐集河畔 高声歌唱生活

我定会孤独返回空无一人的山峦

 

【枫】

日短夜长

路远马亡

 

【五月的麦地】

黄昏常存弧形的天空

让大地布满哀伤的村庄

有时我孤独一人坐在麦地为众兄弟背诵中国诗歌

没有了眼睛也没有了嘴唇

 

【麦地（或遥远）】

谷仓深处倾吐一句真挚的诗

亲人的询问

 

【麦地与诗人】

你无力偿还

一颗放射光芒的星辰

在你头顶寂寞燃烧

…

 

麦地

别人看见你

觉得你温暖，美丽

我则站在你痛苦质问的中心

被你灼伤

我站在太阳 痛苦的芒上

 

【幸福的一日——致秋天的花楸树】

我无限地热爱着新的一日

今天的太阳 今天的马 今天的花楸树

使我健康 富足 拥有一生

 

【重建家园】

为了生存你要流下屈辱的眼泪

来浇灌家园

 

【十四行：夜晚的月亮】

太阳把血

放入灯盏

 

【十四行：玫瑰花园】

四川，我诗歌中的玫瑰花园

那儿诞生了你——像一颗早晨的星那样美丽

 

【秋】

时日漫长 方向中断

动物般的恐惧充塞着我们的诗歌

 

【秋日黄昏】

切开血管

落日殷红

 

【秋】

该得到的尚未得到

该丧失的早已丧失

 

【为什么你不生活在沙漠上】

你要把事业留给兄弟 留给战友

你要把爱情留给姐妹 留给爱人

你要把孤独留给海子 留给自己

 

【祖国（或以梦为马）】

我要做远方的忠诚的儿子

和物质的短暂情人

和所有以梦为马的诗人一样

我不得不和烈士和小丑走在同一道路上

…

 

骑着五千年凤凰和名字叫“马”的龙——

我必将失败

但诗歌本身以太阳必将胜利

 

【秋天的祖国——致毛泽东，他说“一万年太久。”】

心还张开着春天的欲望滋生的每一道伤口

…

 

大地似乎苦难而丰盛

 

【眺望北方】

我要成为宇宙的孩子 世纪的孩子

挥霍我自己的青春

然后放弃爱情的王位

去做铁石心肠的船长

 

【夜色】

我有三次受难：流浪、爱情、生存

我有三种幸福：诗歌、王位、太阳

 

【太阳和野花——给AP】

该忘记的早已忘记

该留下的永远留下

…

 

总是有寂寞的日子

总是有痛苦的日子

总是有孤独的日子

总是有幸福的日子

然后再度孤独

 

【四行诗】

一个美好少女

在月光照耀的地方

说 好好活吧 亲爱的人

 

【冬天】

在黑夜里为火写诗

在草原上为羊写诗

在北风中为南风写诗

在思念中为你写诗

 

【远方】

那时我在远方

那时我自由而贫穷

…

 

远方的幸福 是多少痛苦

 

【黑翅膀】

飞回北方 北方的七星还在北方

只不过在路途上指示了方向，就像一种思念

她长满了我的全身 在烛光下酷似黑色的翅膀

 

【西藏】

西藏，一块孤独的石头坐满整个天空

…

 

没有任何泪水使我变成花朵

没有任何国王使我变成王座

 

【青海湖】

我看见你从太阳中飞来

蓝色的公主 青海湖

我孤独的十指化为天空中雪白的鸟

 

【绿松石】

你是年老的国土上

一位年轻的王，老年国王会伏在你肩头死去

土地张开又合拢

 

【山楂树】

今夜我不会遇见你

今夜我遇见了世上的一切

但不会遇见你

 

【日记】

今夜我只有美丽的戈壁 空空

姐姐，今夜我不关心人类，我只想你

 

【遥远的路程】

一双寂寞的黑眼睛多想大雪一直下到他内部

 

【面朝大海，春暖花开】

从明天起，做一个幸福的人

喂马，劈柴，周游世界

从明天起，关心粮食和蔬菜

我有一所房子，面朝大海，春暖花开

 

【黎明（之一）】

山岗上天空望不到边

山岗上天空这样明亮

我永远是这样绝望

永远是这样

 

【黎明（之二）】

泉水白白流淌

花朵为谁开放

永远是这样美丽负伤的麦子

吐着芳香，站在山岗上

…

 

荒凉大地承受着更加荒凉的天空

 

【四姐妹】

所有的风只向她们吹

所有的日子都为她们破碎

…

 

这是绝望的麦子

请告诉四姐妹：这是绝望的麦子

永远是这样

风后面是风

天空上面是天空

道路前面还是道路

 

【日落时分的部落】

北京啊，你城门四面打开，内部空空

在太平洋的中央你眼看就要海水灭顶

 

【桃花时节】

翅膀血红，富于侵略

那就是独眼巨人的桃花时节

…

 

我俩在高原 在命运三姐妹无声的织机

织出的牧场上相遇

 

【春天】

春天空空荡荡

培养欲望 鼓吹死亡

…

 

我进入太阳

粗糙而光明

 

【拂晓】

我早就说过，断头流血的是太阳

所有的你都默默流向同一个方向

 

【月全食】

铁打的人也在忍受生活

铁打的人也风雨飘摇

所有的道路都通向天堂

只是要度过路上的痛苦时光

那一天我正走在路上

两边的荒草，比人还高

 

【春天，十个海子】

春天，十个海子全部复活

在光明的景色中

嘲笑这一个野蛮而悲伤的海子

你这么长久地沉睡究竟为了什么？

…

 

在春天，野蛮而悲伤的海子

就剩下这一个，最后一个

这是一个黑夜的孩子，沉浸于冬天，倾心死亡

不能自拔，热爱着空虚而寒冷的村庄

 

【献诗】

今天，太阳的新娘就是你

太平洋上唯一的人，远在他方

…

 

黑夜从大地上升起 遮住了天空

 

【黑夜的献诗 献给黑夜的女儿】

天空一无所有

为何给我安慰

丰收之后荒凉的大地

人们取走了一年的收成

取走了粮食骑走了马

留在地里的人，埋得很深

 

 

2015.1.27增补

【上山的孩子】

我的愿望是在最高的峰顶

放一块石头

我要参加山的创造

 

【年轻的山群】

她用手帕束起黑发，也要我们用光明放逐阴影

即使黑夜降临

就在黄昏借夕阳把它焚毁

 

【期待】

追随鸽哨

让自己消融于渐渐蔚蓝的天空

 

【纸鸢】

你自由的程度

等于线的长度

挣脱了，也有一条未蜕化的尾巴

 

你以为是在放牧白云

谁知是风放牧你

 

【小站】

我年轻很小

不用向谁告别告别

有点感伤

我让自己静静地坐了一会儿

…

 

向你们诉愿

是自自然然的事

我要到草原去

去晒黑自己

晒黑日记蓝色的封皮

 

【小叙事】

无论如何

我要留一个形象给你

于是我头戴

各色野花

跑进你梦中

 

我的踌躇

铺成你清晨起来

不曾留意的那条小道

很自然地

你顺着它走下去

写些激动人心的故事

 

【琴】

一年一度李子打头

一直平常的我

如今更平常

 

【岁月】

横拿月亮拨开木叶你走来

…

 

千秋，我怎么记得住

已经过去的一千个秋天

 

【诗经中的两个儿子及其他】

你不用算命

命早就在算你

 

【生日颂（或生日祝酒词）】

我会想到你——我的母亲

在一个冬天 怎样羞涩而温清地

向父亲暗示 你怀了孕

一个生命在腹中悸动

…

 

痛苦是酒精

我们全都沉浸其中

只是分给每个人的酒杯不同

…

 

至于我 早就想成为父亲

虽然我没有妻子

要说有 五六年前就已经结婚

我的妻子就是中国的诗歌 汉语的诗歌

我要成为一首中国最伟大诗歌的父亲

像荷马是希腊的父亲 但丁是意大利之父 歌德是德意志的父亲

我早就想成为父亲 我一定能成为父亲

成为父亲总是人类最大的幸福
