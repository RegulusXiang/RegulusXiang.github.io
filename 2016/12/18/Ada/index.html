    Parameter Passing

    Parameter passing determines how the formal parameters are associtaed with actual parameters

    4 ways of passing

    1. pass by value
    • The formal parameters are distinct valuables (memory locations) from the actual parameters.
    • The value of each actual parameters is copied to the formal parameters.
    1. pass by reference (actual parameters will change)
    • The formal parameters is a reference to the memory location of the corresponding actual parameters.
    • Implemented by passing address (pointer) of the actual parameters into the formal parameters.
    • Requeres an automatic de-reference every time the formal parameters is accessed.
