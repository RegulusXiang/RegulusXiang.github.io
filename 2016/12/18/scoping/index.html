    Binding

    association between a name and what the name represents.

    variable name -> memory location -> value

    type name -> type

    Environment

    set of bindings that hold of a given point in the program

    Scope

    the portion of the program in which the name is visible

    Block

    a syntactic construct that define the Scope

    Block Structure Languages

    • A PL in which blocks can be nested
    • including nested procedures

    Example:

    A reference to a variable is resolved according to the innermost surrounding Scope

    Quiz:

    Almost all modern languages will print 5.

    Static Scoping

    The body of a procedure is evaluated in the environment in which the procedure is defined

    Dynamic Scoping

    The body of a procedure is evaluated in the environment in which the procedure is called.

    Example 2:

    print 25.

    Module

    a portion of code with an interface that restricts accross to the code from the rest of the program.

    Ada

    Ada modules are called packages

    • body
    • specification (interface)

    Thread

    a sequence of operations or events

    2 threads are executed concurrently if no assumptions about the relative order of events between the threads can be made.
