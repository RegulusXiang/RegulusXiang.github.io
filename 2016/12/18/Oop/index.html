    A language is object-oriented has following features:

    Encapsulation of data with the code operates on that data

    e.g.

    Inheritance

    • define a new type based on the definition of an existing class
    • the new type includes all the elements of the existing type

    Subtypes with dynamic dispatch

    • a child type is a subtype of a the parent type, so any object at the child type can be treated as if it were an object at the parent type

    Dynamic Dispatch (Dynamic Overloading)

    • when a method for an object is called, which method it is depends on the actual type of the object, not the decleared type

    A type is a set of values.

    A subtype is a subset of the set of values of the parent type.

    Subset interpretation of subtyping.
