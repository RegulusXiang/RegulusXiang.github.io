    Context Free Grammar(CFG)

    Contains:

    • set of terminal symbols

    • set of non-terminal symbols

    • one of the non-terminal symbols is dediganted the start symbols

    • set of production rules form:

      A -> a, where A is a non-terminal symbol and a is a symbol of the empty string

    Stages of a compiler

    1. lexer (lexical analysis)
      reads in indivisual characters and produces tokens
    2. parser (syntactic analysis)
      takes the stream of tokens from the lexer and builds a parse tree
    3. Type checker
      ensures that types are being used correctly
    4. Code generator
    5. Optimization
