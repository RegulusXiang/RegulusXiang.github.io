    Subtyping

    B $\to $Int denotes the set of all functions that can be applied to a B object

    A $\to$ Int denotes the set of all functions that can be applied to an A object

    • but these can be also be applied to a B object

    So A $\to$ Int $\subseteq$ B $\to$ Int
